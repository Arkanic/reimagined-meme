Changelog

Contains relevant information for every major build.

pre-A7 25/8/21

  • Added a barrel, which is a static object the player can collide with.
  • Re-organized some serverside code.

pre-A6 22/8/21

  • Added site favicon.
  • Player sprite is no longer a square cat photo.
  • Finally fixed movement/interpolation so everything appears buttery smooth, and keyboard controls are no longer a glitchy mess.
  • Added some proper socket protocol enforcement, meaning that invalid messages will result in the server kicking you.
  • Added "enter" command to open/close chatbox. Typing WASD no longer makes the player move whilst typing in the chatbox.
  • Changed the font size of chat messages, and player usernames.
  • Switched over from homemade crap-ics to matter.js physics engine. Entities now contain a "body" parameter that contains the matter.js object.

pre-A5 21/8/21

Maintenance. Cleaned up some cobwebs that appeared.

pre-A3

Chat bubbles now exist ingame. Whenever you send a message, a chat bubble appears above you, and remains for some time, before disappearing.

Usernames are now displayed ingame below players.

pre-A2

Interpolation system now in place, should make movement look smoother, however lag still seems to be a problem.

pre-A1

Basic server + client, including input, output, keyboard + mouse movement, other players.
